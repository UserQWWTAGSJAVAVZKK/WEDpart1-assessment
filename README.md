Project Overview 
This is my website project for The Daily Loaf Bakery. I chose this because I wanted to create 
something simple and useful for a small business. The bakery is family-owned and sells bread, 
pastries, and cakes. I made 5 pages for this website. 
Website Goals 
• Show people the bakery's opening hours and location 
• Let customers see the menu and prices 
• Allow people to send enquiries for custom cakes 
• Make it easy to find contact information 
What I Included 
• A navigation menu on every page so users can click around easily 
• A homepage with a hero image and welcome text 
• An about us page with the bakery story 
• A products page listing breads, pastries, and cakes 
• An enquiry form for pre-orders 
• A contact page with two Cape Town locations and live Google Maps
